# Vege-Resilience
we integrated complementary satellite observations to systematically quantify global vegetation resilience using two robust recovery-rate metrics. We then coupled these multidimensional resilience indicators with global plant functional traits (CWM) to unravel how individual traits and their coordinate trade-offs govern resilience
